# 엑셀 파일 자동 취합 및 마스킹/강조 프로그램

엑셀 파일들을 자동으로 취합하고, 개인정보를 마스킹 처리하며, 중요 키워드를 강조 표시하는 VBA 매크로 프로그램입니다.

## 📋 프로그램 기능

### 1. 자동 파일 취합
- 같은 폴더 내 `_파일취합_` 키워드가 포함된 엑셀 파일들 자동 감지
- `민원데이터` 시트의 데이터를 하나의 통합 시트(`취합`)로 복사
- 첫 파일의 제목행만 포함, 이후 파일은 데이터만 추가

### 2. 개인정보 마스킹
- `민원인명` 열의 이름 자동 마스킹 처리
- 마스킹 규칙:
  - 2자: `김*`
  - 3자: `김*순`
  - 4자: `김**순`
  - 5자 이상: `김***순`

### 3. 키워드 강조
- 다음 키워드를 자동으로 감지하여 빨간색 굵은 글씨로 강조:
  - `긴급`
  - `민원`
  - `주의`

## 🚀 사용 방법

### 전제 조건
- Microsoft Excel (VBA 지원 버전)
- 매크로 실행 가능 권한

### 실행 단계

1. **파일 준비**
   - 취합할 엑셀 파일들의 이름에 `_파일취합_` 추가
   - 예: `데이터_파일취합_001.xlsx`, `정보_파일취합_002.xlsx`
   - 각 파일에 `민원데이터` 시트 필수 포함

2. **매크로 실행**
   - `CombineExcelFilesAndFormat.bas` 파일을 엑셀 VBA 편집기에 임포트
   - 또는 매크로 포함 엑셀 파일 내에 코드 복사
   - `Sub CombineExcelFilesAndFormat()` 실행

3. **결과 확인**
   - 새로운 `취합` 시트 생성 (또는 기존 시트 초기화)
   - 모든 데이터 통합
   - 개인정보 마스킹 완료
   - 키워드 강조 적용

## 📝 주요 특징

✅ **상대 경로 적용** - 매크로 파일 기준 경로 사용으로 어디서나 실행 가능  
✅ **원본 보호** - 원본 파일은 읽기 전용으로 열기 (변경 방지)  
✅ **자동화** - 수동 작업 최소화, 반복 작업 자동 처리  
✅ **보안** - 개인정보 자동 마스킹으로 데이터 보호  
✅ **가시성** - 중요 정보 자동 강조로 빠른 확인 가능

## ⚠️ 주의사항

- 매크로 실행 전 중요 데이터 백업 권장
- 엑셀 보안 설정에서 매크로 실행 허용 필요
- 원본 파일명에 반드시 `_파일취합_` 포함
- 각 원본 파일에 `민원데이터` 시트 필수
